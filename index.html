1. Función implementada con un condicional

Escribí una función recursiva esPotenciade(numero, exponente) que utiliza una estructura if-else-if para ejecutar el caso base y las dos posibilidades.

Recibe un numero y un exponente numéricos.

Devuelve un boolean (true si numero es potencia de exponente o false si no lo es).

Código 
                    
function esPotenciaDe(numero, potencia) {
    // caso base
    if (numero === 1) {
        return true;
    } else if ( numero % potencia === 0) {
        return esPotenciaDe(numero / potencia, potencia);
    } else {
        return false;
    }
}
